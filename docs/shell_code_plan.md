## 目标
学会Shell编程，能读懂别人写的Shell脚本，能独立编写Shell脚本解决工作中的问题。

１. Shell编程是一门永不过时的技术(只要有机器,就不可避免)，是每个高效程序员都应该具备的基本技能。
２. DevOps工作中，很多地方需要用到Shell脚本，学会它能大大提高工作效率。
