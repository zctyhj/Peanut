**管理书籍**
> 《卓有成效的管理者》

**推荐理由**   
> 在这本薄薄的书里，现代管理学之父彼得·德鲁克提出一个重要的定律：管理者的效率，往往是决定组织工作效率的最关键因素。因此，在管理别人之前，你必须学会管理自己。德鲁克也给出了有效管理者必须养成的五种习惯：正确统筹时间；致力于对外界的贡献；重视发挥长处，把工作建立在优势上；集中精力在少数主要领域；做有效地决策。
