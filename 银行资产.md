# 贷款
- 贷款是商业银行的传统核心业务
- 也是商业银行最主要的盈利资产


划分维度
- 权责划分      自营和委托
- 偿还期划分    短期贷款 中期贷款  长期贷款
- 贷款用途      工商业贷款  农业贷款  消费者贷款  有价证券经纪人贷款
- 形态          应记贷款和非应计贷款
- 条件          信用贷款 抵质押贷款  保证贷款  票据贴现
- 质量划分      正常 关注 次级  可疑  损失
